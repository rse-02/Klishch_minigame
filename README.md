# Klishch_minigame
# Цель: Мини-игра змейка

##Задачи:
  1. Рисовать уровень на экране
  2. Захватить нажатия кнопок
  3. Играть музыку

##Ресурсы: 

    1. Кнопки 4
	2. Переключатель
	3. Матрица 8*8
	4. Макетная плата
	5. Arduino
	6. AA Battery 2
	7. Пьеза Динамик

##Состояния:
	1. Ждет
	2. Рисовать уровень
	3. Игра
	4. Конец игры 
	5. Разряжен

## Спосоности:
 ###змейка:
	1. Вправо.
	2. Влево.
	3. Вперед.
	4. Назад.
 ###Устройство
	1. проверить нажатие кнопки 1
	3. проверить нажатие кнопки 2
	4. проверить нажатие кнопки 3
	5. проверить нажатие кнопки 4
	6. показать пиксель
	7. очистить матрица
	8. фрукт появиться
	9. фрукт исчезнуть
